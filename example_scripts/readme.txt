Directory containing scripts which generate the output files for CW distances.

find_upskel_distances.py - example script which generates a KDtree for a set of CW features, enabling distance computation for a queried set of points.

plot_web.ipynb - example notebook to create 3D plots of CW features.

/generate_output_upskel - generates output files containing CW distances for all FoF haloes and subhaloes.
